# Lead-Harvest Prompt

**Where to run:** paste this into each of the 8 *finished* Wave-1 deep-research chats (one at a time), in the same thread that produced the report — so it can draw on everything that thread surfaced, not just the final report.
**What to do with the output:** save to `corpus/reports/leads/NN-tradition-leads.md`.
**Why:** the Wave-1 prompts predate the standard report's "section G (leads)". This recovers the follow-up value those threads found. It is a *lead audit only* — do **not** run new heavy research here (the next reports come from fresh, scoped chats).

```
You've completed a deep-research report for "A Map for Mortals" (a project mapping
human wisdom as recurring forks inside one versioned graph). Do NOT produce a new
research report now. Instead, audit your own coverage and emit LEADS for later waves.

Produce a concise, structured list:

1. COVERED — the thinkers, works, traditions, and dilemmas this report covered well
   (one short line each). This becomes a "do-not-duplicate" note for future queries.

2. UNDER-COVERED / DESERVES A DEEP PASS — specific thinkers, single works, or themes
   that this survey only touched and that warrant their own focused query later
   (e.g. "Epictetus, Discourses — book-by-book extraction"). For each, one line on why.

3. THREADS WORTH FOLLOWING — interesting connections, tensions, or convergences you
   noticed that point somewhere new (including toward OTHER traditions). One line each.

4. HIGH-VALUE SOURCES — specific primary texts, editions/translations (note public-
   domain vs in-copyright), and reputable scholarly references worth using next time.

5. SUGGESTED NEXT QUERIES — 5–10 crisp, scoped query ideas (one tradition/thinker/text
   or thread per query), ranked by value. Phrase each as a ready research target.

Keep it tight and skimmable. No fabricated sources or quotes — if unsure, say so.
```
